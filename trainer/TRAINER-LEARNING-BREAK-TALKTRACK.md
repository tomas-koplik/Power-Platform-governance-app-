# Trainer Learning Break Talk Track

Facilitator-only material. Do not reference this file from participant agents.

## Timing

Use this after the 10-minute Spec Coach phase and before the Build Coach phase.

## Talk Track

"Pause for a moment. You just described an app with AI, and the conversation had a clear shape. That shape was intentional.

The first concept is an agent. An agent is a set of instructions that shapes how the AI behaves. The Spec Coach was designed to help you describe, not build.

The second concept is prompts. Prompts are not just one message. They can be structured instructions that define boundaries, tone, and workflow.

The third concept is context. What you said, and the spec you created, become context the next agent can use.

The fourth concept is skills. Skills are reusable patterns. In this workshop, one skill keeps apps simple and local, and another applies design guidance.

The fifth concept is MCP. MCP lets the assistant connect to trusted sources. Here, GitHub MCP is used for sample data and design guidance.

That is grounding: the assistant is using specific sources instead of guessing.

The important point is that you did not need to configure those pieces during the exercise. You described what you wanted. The guided system helped shape the result."

## Transition

"Now switch to the Build Coach. It will use your spec to build the app. I will continue demonstrating on screen."
