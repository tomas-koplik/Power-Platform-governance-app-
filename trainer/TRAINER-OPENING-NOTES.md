# Trainer Opening Notes

Facilitator-only material. Do not reference this file from participant agents.

## Goal

Set context before anyone opens VS Code.

Make clear:
- this is not a coding workshop
- participants are learning how to collaborate with AI
- the core pattern is Describe -> Review -> Improve
- the output is an interactive app-like experience, not a static report

## Opening Script

"Welcome.

Today is not a coding workshop. You do not need to know how to write code, read code, or understand developer workflows deeply.

The goal is to experience how you can collaborate with AI to turn an idea into a working app.

We will use a simple pattern: Describe -> Review -> Improve.

First, you will describe the app you want. Then you will review what AI creates. Then you will improve it by asking for focused changes.

The app is the vehicle. The real lesson is how to guide AI clearly."

## Transition to Spec Phase

"Now open VS Code and use the Spec Coach. I will do the same on screen, slowly, so you can follow along. You have about ten minutes. Focus on describing the app, not building it yet."

## Avoid Saying Early

Do not lead with:
- agents
- skills
- MCP
- prompt files
- repo structure
- how the system is wired together

Those concepts are revealed after participants have experienced the Spec Coach phase.
